# maekfei.github.io

Personal homepage of **Mingfei Lu (LuMF)** — PhD student at the University of Technology Sydney.

Live: https://maekfei.github.io/

## Stack

- Pure HTML / CSS / vanilla JS (no build step)
- Hosted on GitHub Pages (via `main` branch root)
- Aurora-gradient + glassmorphism dark theme, with light-theme toggle
- Inter / Playfair Display / JetBrains Mono via Google Fonts; Font Awesome icons via CDN

## Edit

- About / News / Research / Projects / Education are all in `index.html`
- Theme tokens live at the top of `assets/style.css` (`:root` block)
- Interactions (nav highlight, theme toggle, BibTeX copy, reveal-on-scroll) in `assets/script.js`
